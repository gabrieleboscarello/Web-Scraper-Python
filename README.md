
Un semplice progetto Python che estrae le **ultime notizie dal sito ANSA** e le visualizza in un'interfaccia grafica (GUI) realizzata con **Tkinter**.


- Estrazione dei titoli delle ultime notizie da ANSA.it
- Interfaccia grafica con `tkinter`
- Scroll automatico per leggere tutte le notizie
- Layout semplice, pronto all’uso
